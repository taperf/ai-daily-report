### Anthropic Claude Blog — Claude Managed Agents 重大更新：Dreaming、Outcomes、多智能体编排

Anthropic launches **dreaming** as a research preview for Claude Managed Agents. Dreaming is a scheduled process that reviews agent sessions and memory stores, extracts patterns, and curates memories so agents self-improve over time. Also shipping: **outcomes** (rubric-based self-grading, +10% task success on hardest problems), **multiagent orchestration** (lead agent delegates to specialists running in parallel), and **webhooks**.

Anthropic 为 Claude Managed Agents 发布重大更新。**Dreaming**（做梦）是一个定时流程，会回顾 agent 的历史会话和记忆库，提取模式并整理记忆，使 agent 随时间自我改进。同时上线：**Outcomes**（基于评分标准的自我打分机制，在最难任务上提升10%成功率），**多智能体编排**（主 agent 分派任务给并行运行的专家子 agent），以及 **Webhooks** 通知。

Key stats / 关键数据:
- Harvey（法律AI）使用 dreaming 后完成率提升约6倍
- Netflix 用多智能体编排并行分析数百个构建日志
- Spiral 用 outcomes 强制写作质量达标，主 agent 跑 Haiku，子 agent 跑 Opus
- 文档生成质量：docx +8.4%，pptx +10.1%

### Unsupervised Learning Ep 87: Gemini Co-Lead Oriol Vinyals 谈世界模型、RL新领域与持续学习

Host Jacob sits down with **Oriol Vinyals**, co-lead of Gemini (alongside Noam Shazeer and Jeff Dean), right after Google IO. Key topics:

主持人 Jacob 在 Google IO 后专访 **Oriol Vinyals**（Gemini 联合负责人，与 Noam Shazeer、Jeff Dean 并列）。核心话题：

- **World models as a path to AGI**: Google is uniquely focused on world models (vs. other labs betting on code/recursive self-improvement). Oriol argues these aren't at different layers — world models and code intelligence can compound.
- **世界模型通向AGI**：Google 独特地押注世界模型（vs 其他实验室押注代码/递归自我改进）。Oriol 认为这两条路径不矛盾，可以互相增强。

- **Memory as the next reasoning frontier**: Advances in memory will look like reasoning advances in the coming years. Long-context + structured memory is key.
- **记忆是下一个推理前沿**：未来几年记忆的进步看起来会像推理的进步。长上下文+结构化记忆是关键。

- **State of scaffolding**: What persists vs what gets absorbed into models. Oriol's read on what founders are building that will last.
- **Scaffolding 现状**：哪些会持续存在，哪些会被模型吸收。
