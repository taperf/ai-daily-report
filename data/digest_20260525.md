4. LLM Agent后端代码生成的「约束衰减」问题被揭示

arXiv新论文《Constraint Decay: The Fragility of LLM Agents in Back End Code Generation》指出，LLM Agent在生成后端代码时存在系统性的「约束衰减」现象——随着对话轮次增加，模型逐渐遗忘先前设定的约束条件，导致生成代码偏离需求。这一发现对当前火热的AI编码助手（Codex、Claude Code、Cursor等）的可靠性提出了重要警示。

🔗 https://arxiv.org/abs/2605.06445
