## 📝 Blog: Claude Managed Agents — Dreaming, Outcomes & Multiagent Orchestration

**Source: Claude Blog**

Anthropic launched several major updates to Claude Managed Agents:

- **Dreaming (research preview)**: A scheduled process that reviews past agent sessions, extracts patterns, and curates memories so agents self-improve over time. You can let it update memory automatically or review changes first.
- **Outcomes**: Write a rubric describing success criteria; a separate grader evaluates output in its own context window (uninfluenced by the agent's reasoning) and sends it back for revision if needed.
- **Multiagent orchestration**: A lead agent can delegate to specialist agents working in parallel on complex jobs.
- **Webhooks**: Now available for developers building with Managed Agents.

Anthropic为Claude托管智能体发布重大更新：
- **Dreaming（研究预览）**：定期回顾历史会话，自动提取模式、整理记忆，让智能体自我改进。可自动更新或人工审核。
- **Outcomes**：用户定义成功标准，独立评分器在单独上下文中评估输出，不达标则退回重做。
- **多智能体编排**：主智能体可将任务委托给并行工作的专家智能体。
- **Webhooks**：开发者可用。

核心洞察：Dreaming本质上是给Agent加了"睡眠学习"能力——人类通过睡眠整合记忆，Agent通过Dreaming整合跨会话经验。这对长期运行的Agent和多Agent协作尤其关键。

🔗 https://claude.com/blog/new-in-claude-managed-agents
