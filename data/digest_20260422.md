4. RAG 搜对了却答错？德国萨尔大学找到真相（ACL'26）
来源：量子位 | 2026-04-18
RAG（检索增强生成）已成为大模型落地标配，但一个令人困扰的问题是：明明检索到了正确文档，生成结果却出错。德国萨尔大学团队在 ACL 2026 上发表研究，系统性分析了这一「搜对答错」现象的成因，为 RAG 系统的可靠性优化提供了新的理论依据。这对所有基于 RAG 架构的企业级应用（包括各类知识库问答系统）都有直接参考价值。

5. 英伟达发布全球首个开源量子 AI 模型家族 Ising
来源：量子位 | 2026-04-16
英伟达推出开源量子 AI 模型家族 Ising，包含一个 350 亿参数的视觉语言模型（VLM）用于量子处理器校准，以及量子纠错解码模型。校准时间从数天缩至数小时，解码速度提升 2.5 倍、准确率提升 3 倍。黄仁勋称 Ising 是量子机器的「操作系统」，能将脆弱量子比特转化为可扩展的「量子-GPU 系统」。哈佛、伯克利、IonQ 等机构已开始部署。这标志着 AI 与量子计算的融合进入了工程化阶段。
🔗 https://hub.baai.ac.cn/view/53989

6. Brex 开源 CrabTrap：用 LLM 当法官保护 AI Agent 安全
来源：Hacker News | 2026-04-21
美国金融科技公司 Brex 开源了 CrabTrap，一个基于 LLM-as-a-judge 的 HTTP 代理，专门用于在生产环境中保护 AI Agent 的安全。随着 Agent 在实际业务中的部署加速，如何防止 Agent 执行恶意请求、越权操作成为关键挑战。CrabTrap 通过在 Agent 与外部服务之间插入一层 LLM 审核，实时判断请求是否安全。这是 Agent 安全基础设施领域的重要开源贡献。
🔗 https://www.brex.com/crabtrap
