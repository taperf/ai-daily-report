**Anthropic Engineering: Scaling Managed Agents — Decoupling the Brain from the Hands**

Anthropic's engineering team published a deep technical post on a key architectural insight for building reliable long-running agents: decouple the "brain" (planning/reasoning) from the "hands" (tool execution). The core finding is that agent harnesses encode assumptions about what Claude can't do—but those assumptions go stale as models improve. For example, Claude Sonnet 4.5 would prematurely wrap up tasks that newer models handle gracefully. The post advocates for lighter harnesses that give models more autonomy, letting the "brain" decide when and how to use tools rather than hardcoding workflows. This is directly relevant to anyone building production agent systems.
🔗 https://www.anthropic.com/engineering/managed-agents

**Anthropic Engineering：规模化托管智能体——将「大脑」与「双手」解耦**

Anthropic 工程团队发表了一篇关于构建可靠长运行智能体的关键架构洞察：将「大脑」（规划/推理）与「双手」（工具执行）解耦。核心发现是，智能体框架中硬编码的假设会随模型进步而过时——例如 Claude Sonnet 4.5 会过早结束任务，而新模型则能从容处理。文章主张使用更轻量的框架，赋予模型更多自主权，让「大脑」自行决定何时以及如何使用工具，而非硬编码工作流。这对任何构建生产级 Agent 系统的人都直接相关。
