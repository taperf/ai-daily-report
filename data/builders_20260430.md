### Dan Shipper, Every CEO

Coined the concept of "Codex-native" apps — software designed to be used inside the in-app browser of your coding agent (Codex, Cowork, Cursor). Both human and agent see the same context, can watch what the other is doing. He's using PostHog inside Codex and finds it "SO MUCH better" — Codex writes queries, sees results, and kicks off agents for PRs or DB requests based on insights. His key take: "a browser inside your desktop coding orchestration tool > an agent in your browser."
