### Anthropic Engineering: Scaling Managed Agents — Decoupling the Brain from the Hands

Anthropic工程博客发布了关于Managed Agents架构的深度技术文章。核心思想：将Agent的「大脑」（Claude和它的harness）与「双手」（sandbox和工具）以及「记忆」（session log）彻底解耦。

**Key insights:**
- Agent harnesses encode assumptions about model limitations that go stale as models improve — e.g., "context anxiety" existed in Sonnet 4.5 but vanished in Opus 4.5
- They adopted the OS abstraction pattern: virtualize agent components (session, harness, sandbox) so implementations can be swapped without disturbing others
- Moved from "pets" (single containers you nurse back to health) to "cattle" (interchangeable, disposable units)
- Critical security fix: credentials are never reachable from the sandbox where Claude's generated code runs — Git tokens are bundled at clone time, OAuth tokens stored in external vaults

**关键洞察：**
- Agent的harness会对模型能力做出假设，但这些假设会随模型进步而过时——比如Sonnet 4.5存在的「上下文焦虑」在Opus 4.5中已消失
- 借鉴操作系统设计哲学：将session、harness、sandbox虚拟化为独立接口，可独立替换
- 从「宠物模式」（精心维护单个容器）转向「牛群模式」（可替换、可丢弃的单元）
- 安全设计：确保sandbox中运行的代码永远无法接触到凭证，避免prompt注入攻击升级

🔗 https://www.anthropic.com/engineering/managed-agents

## 🎙️ Podcast

### No Priors: Baseten CEO Tuhin Srivastava — AI推理市场的供需失衡

Baseten（AI推理云）CEO Tuhin Srivastava上No Priors播客，分享了一年30倍增长背后的故事。

**Key takeaways:**
- Baseten expects >$1B revenue this year, grown 30x in one year
- The inference market is in its earliest stages — by inference count, 99% of enterprises haven't come online yet
- Open source models have "crossed a chasm" in baseline capability; RL techniques + post-training for specialized models are now mainstream
- The application layer will survive because companies have unique user signals that frontier labs can't access (e.g., Abridge's clinician edits, workflow data)
- Custom model adoption is "all ahead of us" — enterprises are just now moving from AI tools → closed-source APIs → custom models

**核心要点：**
- Baseten今年预计收入超10亿美元，一年增长30倍
- 按推理量计算，99%的企业还没上线AI——市场才刚开始
- 开源模型已跨越能力临界点；RL+后训练定制模型已成主流
- 应用层会活下来，因为企业拥有大模型厂商无法获取的独特用户信号
- 企业AI采用路径：AI工具 → 闭源API → 自定义模型——当前大多数还在第二步
